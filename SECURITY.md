# Security Policy
read this file to be sure what version you're using and what versions are supported by PlayDL

## Supported Versions
PlayDL only support v1.2.x and newer versions
always check the repo if you cloned it to get the newest updates/fixes

| Version | Supported          |
| ------- | ------------------ |
| 1.2.x   | :white_check_mark: |
| 1.0.x   | :x:                |

## Reporting a Vulnerability
open an issue with the label **bug** to report a vulnerability.
